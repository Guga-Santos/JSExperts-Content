nvm install 19 = Instala a versão 19 do node.
nvm install --lts = Instala a versão long-term support (versão recomendada).
nvm list = Mostra a lista de tags.
nvm use 18 = Usa a versão 18 que já está instalada na minha maquina.
nvm alias default 18 = Define a versão padrão do Node.js para ser a 18.


Criamos o .nvmrc com a versão desejada e digitamos o comando / nvm use / para ativar a versão.

Se quisermos salvar a versão apropriada para rodar o projeto, digitamos / node -v > .nvmrc /
e será criado um arquivo .nvmrc com a versão atual do node.
